Valhalla Bot

September 2024 - Present

A passion project of mine that I have been working on for over a year now. Valhalla bot is a Discord bot for a FC Mobile game community. That I'm a part of called Valhalla. The bot is built using Python and the Discord library. It has all the features that premium bots have but more personalized for our community. Most of the data is stored in a MongoDB database!

Status Live & Active
Key Features

Custom community commands, MongoDB integration for data persistence, and tailored features for the FC Mobile game community.

Technologies Used
  • Python
  • Discord.py
  • MongoDB
